## 🤖 AI Usage Disclosure
In compliance with Stardance guidelines, I would like to transparently disclose the use of AI tools in building Limux OS. AI assistance accounted for approximately 25% of the codebase, remaining strictly under the 30% policy limit. 

AI was specifically utilized for:
* **Syntax Debugging:** Resolving strict JavaScript scope errors and CSS Flexbox rendering bugs.
* **Canvas Mathematics:** Assisting with the 2D canvas column math for the Matrix digital rain visualizer.
* **Boilerplate CSS:** Generating the base WebKit scrollbar overrides and standard HTML structures.

The core OS architecture, window manager state logic, application integrations (like the hardware camera and local storage logic), and overall system conceptualization were entirely human-engineered.
