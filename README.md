# 🖥️ Limux OS

Welcome to **Limux OS**! It is a superb web-based operating system developed entirely from scratch. Featuring a first-class frosted-glass UI and a proper hacker-style terminal, it runs directly in your browser without requiring any heavy installations. 

It comes pre-packaged with daily utility apps and gives a premium desktop experience right on the web. Kindly check it out and explore the features!

## ✨ Key Features
* **Interactive Hacker Terminal:** A fully functional command-line interface glowing in cyan. Try running commands like `help`, `fetch`, `matrix`, and `hack`!
* **Glassmorphism Design:** Beautiful, translucent windows that blur your desktop wallpaper for a modern aesthetic.
* **In-built Utility Apps:** Comes with a working Calculator, an auto-saving Notes application, and a Calendar widget.
* **Audio Player:** An integrated music player to listen to your favourite tracks while you browse.
* **Hardware Camera:** Real-time webcam capture tool (kindly grant browser permissions to test this feature).
* **Draggable Window Manager:** You can easily drag, minimize, maximize, and close apps just like a real operating system.

---

## 🛠️ Tech Stack Used
This project is built using pure frontend technologies. No heavy frameworks or npm modules are required.
* **HTML5** (Structure)
* **CSS3** (Styling, Flexbox, and WebKit Scrollbars)
* **Vanilla JavaScript** (Window management and app logic)
